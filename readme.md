This project is elaborated on E Homburgers project for a EVO pilot.<BR>It uses a ESP32C3 and has a connector for a NMEA2000 devicenet cable.<BR>The zip file is the zipped gerber files to have the PCB made by PCBWay or JLCPCB.<BR>
This prototype uses modules of aliexpress:<BR>
can module https://nl.aliexpress.com/item/1005006068950633.html (do not forget to break the 121 Ohm resistor)<BR>
transmitter https://nl.aliexpress.com/item/1005009235931766.html (has keys programmed)<BR>
M12 5 pins A-mode male devicenet connector https://nl.aliexpress.com/item/1005006301427198.html<BR>
ESP32C3 module https://nl.aliexpress.com/item/1005006599545782.html<BR>
RXB6 433 MHz receiver module https://nl.aliexpress.com/item/1005010486622832.html<BR>
Power module https://nl.aliexpress.com/item/1005012251181657.html<BR>
